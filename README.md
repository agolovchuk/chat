# Chat

## Цель
Создание простого в интеграции и развертывании чата

## Технические требования к проекту:
### Обязательно:
+ чат 1 на 1
+ груповой чат
+ статус сообщения (набирается, отправлен, доставлено, прочитано)
+ обмен файлами
+ Push notifications

### Желательно:
+ редактирование своих сообщений
+ удаление своих сообщений
